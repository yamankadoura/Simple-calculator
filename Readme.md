# 🧮 Simple Calculator

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)

A modern, responsive calculator built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. The application features a clean user interface, dark mode, persistent calculation history, keyboard shortcuts, and local storage support.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Modulo (%)
- √ Square Root
- x² Square Number
- 🌙 Dark / Light Mode
- 💾 Theme saved with Local Storage
- 📜 Calculation History
- 🗑 Clear History
- 📋 Copy Result to Clipboard
- ⌨️ Keyboard Support (Press Enter)
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight

---

## 🛠 Technologies

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Local Storage API
- Clipboard API


```bash
cd simple-calculator
```

```text
index.html
```

in your browser.
